European Central Bank Statistical Data Warehouse

This dataset is a Linked Data version of the data published by the European Central Bank Statistical Data Warehouse.

It was created as part of the LATC project.

Code used to convert the original data to RDF, and to publish it as Linked Data, is available from github

A SPARQL Endpoint is available here

The terms of use for the data are available on ECB's website
